# CRUD-MongoDB


## Que es esto?
  Es mi primer CRUD realizado con:
    -NODEjs
    -Express
    -EJS
    -MongoDB
    -Mongoose
    
    
    
##Intalar dependencias
```
npm install
```

##Configurar
Debe crear un archivo **.env** para configurar el **user**, **password** y **dbname**

### O localhost
Eliminar la Variable **uri** y colocar el link de localhost

##Ejecutar
```
npm run dev
```
